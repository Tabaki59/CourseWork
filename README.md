 Система "Преподаватель - Студент"

 Что это?
 -----------

 Данный проект представляет собой веб приложени записи студентов на нормоконтроль.
 Выполнен в среде разработки PyCharm (Учебная версия) при помощи фреймворка джанго и
 открытых библиотек языка Python
 
 Последняя версия
 --------------------

 Подробную информацию о последней версии можно найти в данном репозитории

 Требования
 -------------

 Требованияи на стороне сервера является поддержка субд PostgreSQL 
 и достаочные мощности для обработки в пределах 100 рабочих сессий
 Требованиями на стороне клиента будут являться: любой актуальный браузер 
 
 Тестирование системы
 -------------------------

 Проводилось на ПК, операционой системе Windows
 Мобильной версии проект не предполагал

 Установка
 -----------

 Развернуть на сервере файлы проекта, представленные в данном репозитории
